\# Multi-Agent CNN PPO Racing



CNN-based PPO autonomous racing project using Gymnasium CarRacing-v3.



\## Features

\- PPO with CNN Policy

\- Multi-car parallel training

\- Fixed-track training

\- Reward graph generation

\- Checkpoint saving

\- Resume training support



\## Tech Stack

Python, PyTorch, Stable-Baselines3, Gymnasium, PPO
